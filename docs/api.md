Cette documentation inclut une page dédiée à l'API qui affiche automatiquement votre spécification OpenAPI/Swagger.

- Base API: configurée via `customFields.apiBaseUrl`
- Spécification OpenAPI: `customFields.swaggerJsonUrl`

Par défaut, ces valeurs pointent vers `http://localhost:5001` et `http://localhost:5001/swagger/v1/swagger.json`.

Accédez à la page API via le menu « API » ou directement sur l'URL `/api`.
